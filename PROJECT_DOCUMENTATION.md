## 1. Общее описание проекта

Проект представляет собой комплексный **SEO / AEO / GEO аудит** сайтов строительной компании «Тёплый Угол», специализирующейся на строительстве каркасных домов под ключ в Москве и Санкт-Петербурге.

Позже проект расширился до **маркетинговой аналитики**: сводный отчёт по Яндекс.Директ, Calltouch, amoCRM за неделю 15.06–21.06.2026, а также интерактивный HTML-дашборд с KPI, графиками и детализацией по кабинетам и кампаниям.

### Цели аудита:
- Выявить технические блокеры роста органического трафика
- Проанализировать поисковую видимость в Яндексе
- Оценить качество индексации и HTTP-статусы
- Выявить дубли страниц и проблемы с мета-тегами
- Оценить наличие микроразметки (Schema.org, Open Graph, hreflang)
- Проанализировать изображения (alt-теги)
- Сравнить два региональных домена (СПб и Москва)
- Сформировать actionable план работ
- Выделить топ-50 ключевых слов по реальному трафику
- Сформировать 50 потенциальных ключевых слов для расширения семантики

### Цели маркетингового дашборда:
- Визуализировать расход и лиды по кабинетам Яндекс.Директ
- Рассчитать CPA/CPL по кампаниям и аккаунтам
- Сопоставить данные Директа, Calltouch и amoCRM
- Оперативно перераспределять целевые лиды (ЦЛ) между каналами и кампаниями
- Публиковать отчёт на GitHub Pages

### Основные артефакты:
- **`index.html`** — полноценный SEO-отчёт (1284 строки) с 11 разделами, готовый к публикации на GitHub Pages
- **`sk-tu-dashboard-deploy/index.html`** — интерактивный маркетинговый дашборд (984 строки) с Chart.js
- **`marketing_report/week_2026-06-15_2026-06-21/REPORT.md`** — текстовый сводный маркетинговый отчёт

## 4. Скрипты анализа

### 4.1. `get_top50_clicks.py`

**Назначение:** Анализ CSV-файлов Яндекс Вебмастера и суммирование кликов по запросам из обоих доменов.

**Логика работы:**
1. Читает оба CSV-файла из `data/raw/`
2. Для каждого запроса суммирует клики (колонка 4) из обоих доменов
3. Сортирует по убыванию кликов
4. Выводит топ-50 в терминал
5. Сохраняет результат в `data/top50_keywords_by_clicks.json`

**Вывод:**
- Топ-1: «каркасные дома» — 28 952 клика
- Топ-2: «каркасные дома под ключ» — 25 740
- Топ-3: «строительство каркасных домов под ключ» — 15 936

### 4.2. `generate_top50_html.py`

**Назначение:** Генерация HTML-строк таблицы для вставки в `index.html`.

**Логика работы:**
1. Читает `data/top50_keywords_by_clicks.json`
2. Форматирует клики с пробелами как разделителями тысяч (например, «28 952»)
3. Генерирует строки `<tr>` с классами `td-rank`, `td-pos-good`
4. Сохраняет в `data/top50_keywords_html.txt`

**Формат строки:**
```html
<tr><td class="td-rank">#1</td><td><strong>каркасные дома</strong></td><td class="td-pos-good">28 952</td></tr>
```

### 4.3. `find_top_clicks.py`

**Назначение:** Аналог `get_top50_clicks.py`, но только для одного домена `sk-tu.ru`. Используется для быстрой проверки.

### 4.4. `match_clicks.py`

**Назначение:** Сопоставляет устаревший `data/top50_keywords.json` (частота обхода) с реальными кликами из CSV.

**Вывод:** таблица «запрос | частота | клики» и JSON-массив.

### 4.5. `scripts/seo_audit_collector.py`

**Назначение:** Сбор данных SEO-аудита с любого сайта.

**Функционал:**
- Анализ HTTP-статусов
- Проверка мета-тегов (title, description)
- Поиск дублей
- Проверка Schema.org / Open Graph / hreflang
- Анализ изображений (alt)
- Проверка sitemap и robots.txt

**Использование:**
```bash
python scripts/seo_audit_collector.py https://domain.com
```

### 4.6. `scripts/gsc_collect_data.py`

**Назначение:** Сбор данных из Google Search Console через OAuth.

**Использование:**
```bash
python scripts/gsc_collect_data.py --token YOUR_ACCESS_TOKEN --site sc-domain:domain.com
```

### 4.7. `scripts/keyword_analyzer.py`

**Назначение:** Анализ ключевых слов из HTML-страниц и URL-структуры.

**Логика:**
- Извлекает ключевые слова из URL-путей
- Извлекает слова из `<title>` HTML-страниц
- Фильтрует стоп-слова и технические части
- Считает частоту и формирует отчёт

### 4.8. `scripts/extract_keywords.py` / `scripts/extract_keywords_json.py`

**Назначение:** Извлечение ключевых слов из контента сайта.

### 4.9. `scripts/generate_keyword_sections.py`

**Назначение:** Генерация разделов с ключевыми словами для отчёта.

### 4.10. `scripts/analyze.py`

**Назначение:** Общий анализ данных аудита (meta-теги).

### 4.11. `marketing_report/week_2026-06-15_2026-06-21/check_leads.py`

**Назначение:** Подсчёт целевых лидов (ЦЛ) из `source_4_amocrm_leads.xlsx`.

3. Считает статусы
